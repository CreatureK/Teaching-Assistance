# 前端开发环境使用指南

## 项目概述

这是一个基于以下技术栈的前端项目：
- Vue 3
- TypeScript
- Vite
- Element Plus
- Vue Router

## 环境要求

- Node.js (建议使用最新LTS版本)
- Yarn 包管理工具

## 安装步骤

1. 进入项目目录：
```bash
cd frontend
```

2. 使用Yarn安装依赖：
```bash
yarn install
```

## 开发命令

### 启动开发服务器

```bash
yarn dev
```
启动后可通过浏览器访问本地开发服务器（一般为 http://localhost:5173）


## 项目结构

```
frontend/
├── src/                  # 源代码目录
│   ├── assets/           # 静态资源文件
│   ├── components/       # 通用组件
│   ├── hooks/            # Vue Hooks
│   ├── router/           # 路由配置
│   ├── views/            # 页面视图
│   ├── App.vue           # 根组件
│   ├── main.ts           # 入口文件
│   └── style.css         # 全局样式
├── public/               # 不需要打包的静态资源
├── index.html            # HTML模板
├── vite.config.ts        # Vite配置
├── tsconfig.json         # TypeScript配置
└── package.json          # 项目依赖和脚本
```

## 常见问题

### 安装依赖时出现网络问题

可以尝试配置Yarn镜像源：
```bash
yarn config set registry https://registry.npmmirror.com
```

### 添加新依赖

```bash
yarn add 包名       # 添加生产依赖
yarn add -D 包名    # 添加开发依赖
```

### 路径别名

项目已配置`@`路径别名指向`src`目录，可以这样使用：
```typescript
import Component from '@/components/Component.vue'
```
